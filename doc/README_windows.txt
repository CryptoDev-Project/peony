Peony Client
=====================

Intro
-----
Peonys is a Proof-of-Stake digital asset based mainly adopted as cryptoccurrency and paymenth method
Main features are:
- Masternode technology withh See-Saw rewards mechanism
- Governance Bugeting System
- SwiftTX for fast transactions
- Obfuscation protocol 

Setup
-----
Unpack the files into a directory and run peony-qt.exe.

Peony Client is the original desktop wallet and it builds the backbone of the network working and acting as a node and staking wallet.
However, it downloads and stores the entire history of transactions on Peony Blockchain;
According on the speed of your computer and network connection, the synchronization process can take anywhere from a few hours to a day or more, specially all synchronization regarding masternode network.
